    介绍lsblk命令的详细用法。

    lsblk

    功能：列出块设备

    描述：

    用法：

    -a 是默认选项

    head说明

    HEAD 说明
    NAME 块设备名
    MAJ:MIN 主要和次要设备号
    RM 设备是否可移动设备(1为可移动)
    SIZE 设备的容量大小信息
    RO 设备是否为只读(1表示只读)
    TYPE 块设备是否是磁盘或磁盘上的一个分区(disk表示是磁盘，part表示是分区, rom表示只读存储)
    MOUNTPOINT 设备挂载的挂载点

    选项列表

    选项示例

    -b

    以字节为单位而非易读的格式来打印 SIZE

    -d

    不打印从属设备(slave)或占位设备(holder)

    -f

    输出文件系统信息

    -m

    输出权限信息

    -l

    使用列表格式的输出

    -P

    使用 key=“value” 输出格式

    -t

    输出拓扑信息

    其中 SCHED 是调度算法，有 cfq、deadline 等等。

    -S

    输出有关 SCSI 设备的信息

    -o

    输出列

    -n

    不打印标题
