参数 功能
-nc 下载时跳过已经存在的文件
-B 使用 -F -i 文件选项时，在相对链接前添加指定的 URL
-nH 不创建主机名目录
-i 下载从指定文件中找到的 URL
-v 显示信息
-r 递归下载
-l 最大递归深度(inf 或 0 表示无限)
-F 以 HTML 方式处理输入文件
–cut-dir=num 忽略远程目录中指定数目的目录层
-E 将所有 MIME 类型为 text/html 的文件都加上 .html 扩展文件名
