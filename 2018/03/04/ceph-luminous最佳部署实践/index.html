  • ceph

luminous版本ceph最佳部署实践，分别指定rocksdb、osd数据及元数据磁盘位置，最大化集群性能。

磁盘分区

每个OSD对应需要4个磁盘分区（简单起见在一块磁盘上分的四个区）

四个分区分别用于：

分区 功能 磁盘介质 推荐大小
/dev/vdd1 OSD元数据存放 SSD 1G
/dev/vdd2 rocksdb数据存放 SSD
/dev/vdd3 rocksdb wal SSD 5G
/dev/vdd4 实际数据存放 SATA 单块SATA盘

修改配置文件

1、修改配置文件 ceph.conf 中关于 osd 的部分，为每个OSD的不同部分指定磁盘

  • db path为rocksdb 存放数据的位置
  • wal path 为rocksdb的原子操作位置
  • block path 为实际数据存放的位置
  • /dev/vdd1 作为osd的元数据存放位置
